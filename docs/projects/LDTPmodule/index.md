# 生活習慣病療養計画書アプリ

**外来診療を止めない療養計画書の作成管理ツール**

## 概要

生活習慣病療養計画書を効率的に作成・管理する Windows デスクトップアプリ。患者 ID に紐づいて計画内容を SQLite に保存し、次回受診時は前回内容をコピーして修正するだけで完結する。Excel テンプレートを活用してバーコード付きの計画書を即座に生成する。

## 想定ユーザーと使用シーン

- 生活習慣病（糖尿病・高血圧・脂質異常症など）を診る外来医師
- 療養計画書の作成に毎回手間をかけている医療スタッフ

**想定ワークフロー:** 電子カルテから出力した CSV を自動読み込みし、前回内容を呼び出して修正・印刷するだけで計画書が完成する。

## 特徴

1. **患者データの継続管理** — SQLite に患者ごとの計画内容を保存し、前回データをコピーして差分だけ修正できる
2. **バーコード付き Excel 出力** — 公開 Excel テンプレートに python-barcode 生成画像を配置し、印刷・PDF 化まで対応
3. **CSV 自動監視** — 電子カルテ出力の CSV を常時監視し、即座に患者情報を UI に反映する
4. **サーバーレス共有運用** — SQLite ファイルをファイルサーバーに置くだけで複数端末から利用可能
5. **疾病別テンプレート** — 疾病ごとのデフォルト値をテンプレートで管理し、入力を最小化
