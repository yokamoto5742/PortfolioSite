# VisionOCR

**Google Cloud Vision API を使ったデスクトップOCR ツール**

## 概要

VisionOCR は、Google Cloud Vision API を使用して画面上の任意の領域や PDF ファイルからテキストを抽出する Windows 用デスクトップアプリケーションです。マウスドラッグで範囲を選択するだけで高精度の OCR 処理を実行でき、日本語の手書き文字や半角カタカナの認識にも対応しています。

## 想定ユーザーと使用シーン

- スキャンされた PDF や画面上のテキストを素早くコピーしたい方
- 手書き文書や紙のフォームをデータ化したい方

**想定ワークフロー:** 画面またはPDFの対象領域を選択し、OCR結果をUI画面で整形して出力する。

## 特徴

1. **マウスドラッグによる領域選択 OCR** — 画面の任意の矩形領域をドラッグ選択するだけでOCRを実行。スムーズな操作性を実現。
2. **PDF 一括テキスト抽出** — 複数ページの PDF を選択するだけで全ページを自動処理して表示。
3. **日本語特化の高精度認識** — 文章形式・表形式の 2 種類の検出タイプを選択でき、手書き文字や半角カタカナにも対応。
4. **テキスト整形機能** — 読点・句点・改行・空白の除去ボタンで OCR 結果をワンクリックで整形。
5. **追記 / 上書きモード切り替え** — 連続して OCR を行う際の結果の挿入方法を柔軟に制御可能。
