# ManualSearch

**PDF・テキスト・Markdown ファイルの全文検索デスクトップアプリ**

## 概要

ManualSearch は、共有フォルダ上の PDF・テキスト・Markdown ファイルを横断検索できる Windows 向けデスクトップアプリです。検索結果に該当する PDF ページを Adobe Acrobat Reader DC で自動的にハイライト表示し、「あのマニュアルのどこに書いてあったっけ」という悩みをすばやく解決します。

## 想定ユーザーと使用シーン

- 社内マニュアルや手順書を日常的に参照するスタッフ
- 複数フォルダにまたがる PDF 資料を管理する部署
- 特定の記述がどのファイルのどのページにあるかを素早く探したい人

**想定ワークフロー:** キーワードを入力して検索 → 該当ファイルを選択 → Acrobat が該当ページを自動で開いてハイライト表示する。

## 特徴

1. **クロスフォルダ全文検索** — 複数の共有フォルダを一括検索。AND/OR クエリとマルチスレッド処理で高速に結果を返す
2. **インデックス高速化** — ファイル更新日時を利用した差分インデックスで、2 回目以降の検索を大幅に短縮
3. **Adobe Acrobat 自動ハイライト** — 検索結果の PDF を開くと、該当箇所が Acrobat Reader DC 上でそのままハイライト表示される
4. **自動エンコーディング検出** — Shift_JIS / UTF-8 を自動判別し、文字化けなく日本語テキストを検索可能
